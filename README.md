# АЖОПС - Автоматизированная Железнодорожная Оперативная Программная Система

Одностраничное веб-приложение (SPA) для просмотра информации о железнодорожных составах и генерации QR-кодов для различных типов оборудования.

## Функциональность

- Выбор состава из списка доступных поездов
- Просмотр статистики по меткам для выбранного состава
- Выбор типа оборудования (подвагонное, салонное, кабина)
- Генерация QR-кодов для каждой метки оборудования
- Навигация между метками (вручную или в автоматическом режиме)

## Структура проекта

```
├── index.html          # Основной HTML-файл
├── css/
│   └── style.css       # Стили приложения
├── js/
│   └── main.js         # JavaScript логика приложения
└── data/
    ├── trains.json     # Список доступных составов
    └── trains/         # Каталог с данными о составах
        └── EP2D-0002.json  # Пример данных о составе
```

## Технологии

- HTML5, CSS3 для интерфейса
- Vanilla JavaScript для логики приложения
- QRCode.js для генерации QR-кодов
- JSON для хранения данных

## Установка и запуск

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере или разверните приложение на веб-сервере
3. Для разработки можно использовать локальный сервер, например, Live Server в VS Code

## Особенности

- Адаптивный дизайн для мобильных устройств
- Асинхронная загрузка данных о составах
- Интуитивно понятный пользовательский интерфейс
- Автоматический режим переключения между метками каждые 15 секунд

## Скриншоты

- Основной экран выбора состава
- Экран выбора типа оборудования
